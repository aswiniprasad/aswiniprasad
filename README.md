## 🚀 What I'm Building

A portfolio of production platforms across two domains — private aviation and
consumer-brand intelligence — all built on a shared serverless GCP foundation
(Cloud Run · Cloud SQL · Cloud Tasks · Secret Manager · Workload Identity
Federation).

- **Aviation Intelligence Platform** — A multi-tenant analytics platform serving
  several private-aviation operators. Flight-history ingestion from JetNet, FBO
  fuel-price intelligence, aircraft/contact enrichment, and email/SMS outreach —
  one shared backend serving isolated per-client dashboards.
- **FBO Campus Operations & Revenue Analytics** — For a private-aviation hangar
  operator: occupancy, lease-health, and revenue analytics driven by a live
  HubSpot CRM sync, plus real-time air-traffic radar via the OpenSky Network.
- **Spirits Sales & Marketing Intelligence** — For a premium American whiskey
  brand: an idempotent sales/depletions analytics pipeline plus an AI marketing
  engine (Claude + pgvector + Flux/Veo media generation) with built-in
  regulatory compliance review.
- **Outreach** — A multi-product email-campaign platform with **AI email
  drafting** (Claude), organic reputation-aware sending (Cloud Tasks), and
  real-time open/click engagement tracking via SendGrid webhooks.
